# Shreyan Portfolio

A light, interactive portfolio website featuring project displays, a sarcastic roast generator powered by Gemini.

##  Features

- **Interactive UI:** Smooth animations and page transitions.
- **Dark Theme:** A sleek, modern, and minimalist dark design aesthetic.
- **Sarcastic Roast Generator:** An AI-powered feature using the Google Gemini API to roast the user.
- **Project Showcase:** Beautifully displayed personal projects and skills.

##  Tech Stack

- **Framework:** React (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **AI Integration:** Google Gemini API 

## 🛠 Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shreyyyan/shreyan-port.git
   cd shreyan-port
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Gemini API key (if applicable):
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
